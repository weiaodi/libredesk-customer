# LibreDesk 项目介绍

> 原文: https://docs.libredesk.io/introduction
> 关键词: 介绍, 功能, 特性, omnichannel, 客户支持, 开源

## 项目简介

LibreDesk 是一个现代化、开源、可自托管的全渠道（Omnichannel）客户支持平台。内置实时聊天、邮件等渠道，打包为单一二进制文件分发。

- **后端**: Go
- **前端**: Vue.js + Shadcn UI (基于 Radix Vue)
- **官网**: https://libredesk.io
- **在线演示**: https://demo.libredesk.io
- **GitHub**: https://github.com/abhinavxd/libredesk

## 核心功能

### 全渠道收件箱 (Omnichannel Inbox)
实时聊天、邮件等渠道统一在一个收件箱中管理，所有对话从单一界面处理。

### 实时聊天 Widget (Live Chat Widget)
可在网站嵌入实时聊天组件，即时与访客互动，直接从支持平台处理实时对话。

### 细粒度权限控制 (Granular Permissions)
支持为团队和个人 Agent 创建自定义角色，权限粒度精细。

### 自动化 (Automations)
通过自动化规则消除重复任务，可基于自定义条件自动打标签、分配和路由对话。

### CSAT 满意度调查 (CSAT Surveys)
通过自动调查问卷衡量客户满意度。

### 宏操作 (Macros)
将常用消息保存为模板，一键发送预设回复、设置标签等。

### 组织管理 (Organization)
通过标签、自定义对话状态和 Snooze 功能组织对话，搜索栏快速定位任何对话。

### 自动分配 (Auto Assignment)
通过自动分配规则，基于 Agent 容量或自定义条件自动分配对话。

### SLA 管理 (SLA Management)
设定和跟踪响应时间目标，当对话有违反 SLA 承诺的风险时发出通知。

### 自定义属性 (Custom Attributes)
为联系人或对话创建自定义属性，如订阅计划、首次购买日期等。

### AI 辅助 (AI-assist)
即时用 AI 重写回复，使其更友好、专业或更精炼。

### 活动日志 (Activity Logs)
跟踪 Agent 和管理员执行的所有操作（更新和系统关键事件），用于审计和问责。
