# Go 语言概述与项目全景

> 本系列文档以 LibreDesk（开源客户支持平台）的真实代码为教学实例，系统讲解 Go 语言的核心基础概念和主流技术栈。所有示例均来自项目源码，标注了对应的文件路径，方便对照阅读。

---

## 什么是 Go？

Go（又称 Golang）是 Google 于 2009 年发布的开源编程语言，设计目标为：

- **编译速度快**：编译型语言，无需解释器或虚拟机
- **并发原生支持**：goroutine + channel，写并发程序像写顺序代码一样简单
- **极简语法**：关键字仅 25 个，代码风格统一
- **强大标准库**：HTTP、JSON、加密、文件系统等开箱即用
- **部署简单**：编译为单一二进制文件，无外部依赖

## LibreDesk 项目技术栈概览

LibreDesk 是一个功能完整的客户支持平台，Go 后端使用了以下主流技术：

| 技术领域 | 选用方案 | 项目中的用途 |
|---------|---------|------------|
| HTTP 框架 | `fasthttp` + `fastglue` | 高性能 HTTP 服务与路由 |
| 数据库 | PostgreSQL + `sqlx` | 数据持久化与查询 |
| 缓存/会话 | Redis + `go-redis` | 会话管理、限流、缓存 |
| 配置管理 | `koanf` | TOML 配置文件 + 环境变量 |
| 日志 | `logf` | 结构化日志 |
| WebSocket | `fasthttp/websocket` | 实时消息推送 |
| 认证 | JWT + Session + OAuth2 | 多方式用户认证 |
| 国际化 | `go-i18n` | 多语言支持 |
| SQL 管理 | `goyesql` | SQL 语句与代码分离 |
| 文件嵌入 | `stuffbin` / `embed` | 静态资源打包进二进制 |
